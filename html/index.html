LED Strip Test

Make sure that you've connected it right:

  1. Power: must be powered by at least 5V 3A DC supply
  2. Polarity: make sure that you've connected +5v to +5v and GND to GND
  3. Data direction: make sure that data wire arrow flows from controller
  4. Wire length: It is not recommended to use wires between controller and LED longer than 1m (3ft)

During this test your LED strip will light each color for 2 seconds.
